Function expression

  • Harus didefinisikan terlebih dahulu sebelum dipanggil
  • Lebih powerfull:
    • Sebagai closure
    • Sebagai argumen untuk Function lain
    • IIFE (Immediately Invoked Function Expression

Contoh
